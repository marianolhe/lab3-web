The Shattered Crystal of Arendall

Description
This project is an interactive fantasy story developed for the Web Systems and Technologies course. The story follows the journey of a former apprentice of the Crystal Guardian who must make decisions to restore the balance of the kingdom of Arendall. 
Each choice leads to different outcomes, including good, bad, and neutral endings.
The main focus of this laboratory is the implementation of modern CSS techniques, responsive layouts, and a consistent visual theme using CSS variables.

Project Structure
The project is organized as follows:
pages/
Contains all HTML files grouped by sections (intro, decisions, endings).
css/
story-theme.css – Contains design tokens and color variables.
main.css – Contains layout, typography, and components.
animations.css – Contains keyframe animations and microinteractions.
assets/
Contains images and fonts used in the project.
